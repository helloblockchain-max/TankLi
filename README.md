# 《战区阿尔法：二战装甲风暴》(Warzone Alpha: WWII Armor Storm)

## 游戏定位与核心架构
*   **类型**：单人网页版俯视角射击游戏（Web-based Top-down Shooter）。
*   **部署方式**：纯前端架构（HTML5 Canvas +原生 JavaScript），通过 **GitHub Pages** 自动化部署发布。
*   **品牌扉页**：游戏加载进入主菜单时，显眼展示：**“为爸爸的马蛋制作”**。

## 资产配置池：双轨制坦克选择与升级树
两款不同逻辑的坦克供玩家首发选择，代表不同的“投资策略”与风险偏好：

### 🛡️ 选项A：“虎式”重型装甲车（价值投资流 / 稳健流）
*   **初始面板**：高血量、高单发伤害、低移动速度、低射速。
*   **操作手感**：步步为营，依靠重装甲硬扛火力，一发入魂。
*   **专属升级路径**：
    1.  **倾斜装甲锻造**：提升基础“跳弹率”。
    2.  **88mm穿甲高爆弹**：击中产生溅射伤害（AOE）。
    3.  **战地维修系统**：缓慢回复生命值。

### ⚡ 选项B：“谢尔曼”高机动战车（成长投资流 / 机动流）
*   **初始面板**：低血量、低单发伤害、极高移动速度、高射速。
*   **操作手感**：跑酷走位，在敌军炮火缝隙中高频输出。
*   **专属升级路径**：
    1.  **双联装机枪/副炮**：小口径开火，形成密集弹幕。
    2.  **引擎涡轮超载**：获取短暂的“冲刺”位移技能。
    3.  **穿甲钨芯弹**：极高概率打穿重型坦克装甲，按比例扣血。

## 核心风控机制：跳弹系统（Ricochet System）
*   **运算逻辑**：当炮弹接触坦克时，系统计算【装甲防御】+【受击角度】的概率，有一定概率触发“跳弹”，实现完全免伤。
*   **视觉反馈**：触发跳弹时，爆出金属火花特效，伴随“叮”的音效，并提示 `RICOCHET!`。

## 剧情主线（五大关卡大逃杀）
*   **序章**：“指挥部失去了联系。物资匮乏，四面楚歌。生存下去，为了爸爸的马蛋，发动引擎！”
*   **第一关：破晓突围**。晨雾废墟，对抗轻型战车，积累初代资金。
*   **第二关：夺桥遗恨**。狭窄石桥争夺战，对抗隐蔽反坦克炮。
*   **第三关：钢铁丛林**。泥泞森林，树木可破坏，对抗中型坦克群包抄。
*   **第四关：风雪阿登**。冰雪湖面（打滑机制），极端天气对抗重型虎王坦克。
*   **第五关：帝国毁灭**。钢铁兵工厂内部，对抗终极多炮塔BOSS坦克。
